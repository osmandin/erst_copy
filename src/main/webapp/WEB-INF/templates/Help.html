Electronic Records Submission Tool

The Electronic Records Submission Tool was developed by IT Services using code created by Tufts University under a grant from the National Historical Publications and Records Commission (NHPRC).

This document describes steps for users from departments and units (external to the University Archives) using the Electronic Record Submission Tool website. The general workflow is that a user completes an online Submission Agreement Form. Upon receipt, the information is entered into the Electronic Records Submission Tool by University Archives staff. When completed, the staff will notify the office. Departments/Unit representatives will then use the website tool to create transfer requests of electronic records. These requests will then be reviewed and approved by Archives staff and files moved to the Trusted Digital Repository pre-ingest area.

Note: A Submission Agreement need only be created once per office.

1.0 Instructions for Units and Offices

1.1. Login

From the main page (Figure 1), users login to access the tool using their username. Login is only possible after completing a Submission Agreement form, see 1.2. NOTE: Login is not required for submitting a Submission Agreement Form.

1.2 Submit Submission Agreement Form

A Submission Agreement Form (Figure 2) can only be completed prior to login. The option is not available after a user logs into the website.

1. Under First time Users, Select the Submission Agreement Form link.

2. The Submission Agreement Form - Verify appears. Enter the Captcha text.

3. Carefully review the online document and then fill in fields for Department/Unit, Department Address, Contact Name, Contact Email, Contact Phone, and the name of the Head of Department/Unit.

4. At the bottom, enter your name to acknowledge you have read the information provided. Users can Print or Save a copy of this agreement.

5. Select Submit Submission Agreement Form. An email will be generated and sent to the University Archives. A copy will also be sent to the email address specified in the form.

Upon notification of a new Submission Agreement form, University Archives staff will use the information to complete the Submission Agreement. The Submission Agreement defines the type of information being sent to the archives and is used as a template for electronic record transfers. University Archives Staff will also add the user account to the tool.

Figure 2: Submission Agreement Form

1.3 Submit Records to Archives/Transmittal Form

Once a Submission Agreement has been entered by the University Archives staff, departmental and unit users can submit records to the Archives. Login (section 1.1) is required.

1. After logging in to the website, the Electronic Records Submission Tools page is displayed. Figure 3)

2. Select Submit Records to Archives/Transmittal Form link. The Submit Records to Archives page is displayed (Figure 4).

3. Select your office from the list to begin creating your transfer request.

4. Select Submit.

5. The Records Submission Form is displayed to the user (Figure 5). Review the information presented at the bottom of this web page under Submission Agreement Information (Figure 6). If anything is incorrect, please contact the archive Do not proceed with the transfer. Select Logout. If the information is accurate, proceed with completing the form.

Figure 5: Records Submission Form

6. Enter the Date span of the records being sent. For example Beginning Year 2014 Ending Year 2015.

7. For Description/Comments, provide any additional information, excluding an inventory, about the history of the records, the person or office that created the records, or any special considerations regarding the records such as physical condition and arrangement/organization. In most cases, this will be left blank.

8. Select Next. The Add digital content files page is displayed (Figure 7).

9. Select Browse. Navigate to directory and select one or multiple files. Use CTRL (PC) or CMD (Mac) to select multiple files or Shift Key for range. When complete, Select Open.

10. The files selected will be displayed along with their size. If you inadvertantly selected an incorrect file, select Clear and Choose Files again. An Inventory is automatically generated. Users can create an inventory and select during the file selection step.

Figure 7: Add digital content files

11. To select additional files and/or from a different folder, select the +. Repeat steps 9 and 10. Note: a maximum of 50GB can be sent per transfer. If more than 50GB is needed, send additional transfers.

12. Select Submit.

13. This completes the submission and an email is sent to the sender and the Archives staff.
